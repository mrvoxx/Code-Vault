IDs and Classes (Simple Explanation)

Example of ID

This paragraph has an ID: id="uniqueText" (Only ONE element in the page can use this ID)

Example of Class

This is a class example: class="infoText"

This is another element using the SAME class.

You can reuse classes as many times as you want.

ID vs Class Differences

  • ID is unique (only one element can have it)
  • Class can repeat (many elements can share it)
  • ID = identify a single element
  • Class = group similar elements
